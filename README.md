Student Record Management Interface
Overview
This project is a simple system for managing student records. It allows users to add, view, update, and delete student information.
Features
•	Add student records
•	View student list
•	Update student details
•	Delete student records
Technologies
•	JavaScript
How to Run
1.	Download or clone the project
2.	Open index.html in your browser
Purpose
This project was created to learn how to manage data and build a simple user interface
